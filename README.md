# 프로젝트 타이틀

수업 중 진행한 리액트 API 연동 실습

## 프로젝트 정보

1. 기존 실습에서 loading 페이지, 영화 카드 구성 CSS, Link 및 Component 구성을 커스텀했습니다.
2. API 데이터를 가져오는 시간을 고려하여 비동기 작업을 사용했습니다.

### 개선 방안

1. 영화 검색 기능 추가
2. 로딩 페이지의 간격 조정 또는 조금 더 효율적인 로딩 페이지(영화와 관련된 or 시선을 고정시킬 수 있는 로딩 페이지)
